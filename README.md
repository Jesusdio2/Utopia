# A recompiled version of Utopia.

Original video: https://www.youtube.com/watch?v=Zy9N48udcdc&t=449s

Original repository: https://github.com/carykh/Utopia

Download Processing here! https://processing.org/download

Controls: Press WASD + Space to move around. Press 'C' to have the camera start following the highlighted creature! (The one blinking white that is nearest to you.)
